N-\/cube Magic \mbox{\hyperlink{class_cube}{Cube}} is consists of 6 sides in different colors\+:~\newline
 White Yellow Red Orange Blue Green Each side is consists of 9 elements (same colors as the sides)~\newline
 ~\newline
 Rules\+: User can rotate all layers\+: vertical, horizontal, front \begin{DoxyWarning}{Warning}
if size of cube is an odd number, user can\textquotesingle{}t rotate central layer (of all the sides), but can give the command (it\textquotesingle{}s will be ignored)~\newline
 ~\newline
 Commands User can give the command in format\+: command$\vert$is\+Inverse$\vert$layer\% ~\newline
 Available commands\+: ~\newline
 V -\/ rotate one of the vertical layers ~\newline
 H -\/ rotate one of the horizontal layers ~\newline
 P -\/ rotate one of the layers in the frontal plane ~\newline
 V -\/ rotate the zero layer vertically (is equal to V0) ~\newline
 P2 -\/ rotate the second layer in the frontal plane H\textquotesingle{}2 -\/ rotate the second layer in the frontal plane (anticlockwise)
\end{DoxyWarning}
